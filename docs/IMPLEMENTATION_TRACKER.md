## Changelog

### 2026-07-13

**Phase 3.1 — Project Scaffold**
Status: Completed

Initialized pnpm monorepo with Turborepo. Scaffolded `packages/shared` (TypeScript types for Service, Artisan, Testimonial, Booking, Availability, GiftCard, ApiResponse; constants for booking-steps, service-categories, animation-timing). Scaffolded `packages/db` (Drizzle ORM schema definitions, migrations config). Scaffolded `packages/api` (Express server with middleware stack, health check endpoint). Scaffolded `apps/web` (Vite 6 + React 18 + TypeScript strict mode, Tailwind CSS v4 with CSS-first `@theme` blocks, TanStack Router with file-based routing, TanStack Query, Redux Toolkit). Configured shared tooling (ESLint, Prettier, tsconfig presets, Husky + lint-staged). All packages verified: TypeScript compiles clean, ESLint passes, dev server starts.

---

**Phase 3.2 — Application Shell**
Status: Completed

Built the Application Shell infrastructure (20 items). Implemented Root Layout (`__root.tsx`) with provider composition (Redux → Query → Theme → Animation → Portal → Cursor). Created Theme Provider with reduced-motion detection. Created Animation Provider with Lenis smooth scroll initialization and GSAP ScrollTrigger integration. Created Portal Provider for modal and toast roots. Created Cursor Context with `fine` pointer detection. Implemented Error Boundaries (Root catastrophic + Section local fallback). Created Loading Indicator (warm gold pulse, no spinner). Built SEO Head component with per-route metadata. Created Skip-to-Content accessibility link. Implemented Scroll Manager with hash deep linking and IntersectionObserver section tracking. Built Web Vitals monitoring (LCP, INP, CLS, TTFB via PerformanceObserver). Built Performance Monitor (long task tracking, memory monitoring). Created Global CSS (tailwind.css with full `@theme` tokens, fonts.css with `@font-face` declarations, animations.css with keyframes, print.css). All verified: TypeScript clean, ESLint clean, dev server HTTP 200.

---

**Phase 3.3 — Layout Skeleton**
Status: Completed

Built 18 layout primitives per the Design System. Created AppLayout (root wrapper with bg-surface, text-text, font-kerning). Created PageContainer with SafeArea integration and configurable element type. Created SectionContainer with 5-tier spacing tokens (intimate/personal/social/formal/public). Created SectionWrapper with background and paddingY controls. Created MaxWidth with 4 width presets (narrow 65ch / content 60-65% / wide 90% / full). Created ContentGrid with responsive CSS Grid variants (auto/two/three/asymmetric) using Tailwind responsive classes (md: 2-col, lg: 3-col, lg: 2fr/1fr). Created Stack flexbox primitive with direction, gap, align, justify, wrap. Created Spacer with token-based sizing. Created VisuallyHidden (WCAG-compliant clip-rect for screen readers). Created Divider (functional-only per VISUAL_RULES N29 — never decorative). Created SkipLink (extracted to reusable component). Created SafeArea with CSS env() inset support. Created ViewportHeight utility (svh/dvh/vh modes for mobile address bar). Created ScrollOffset for sticky nav compensation. Created Semantic Landmarks (Header, Main, Footer, Nav). Created Print Stylesheet (essential content only, no animations, no overlays). Created barrel export index.ts. Updated main.tsx to import print.css. Fixed ContentGrid responsive styles (inline → Tailwind responsive classes). Fixed duplicate identifier errors in MaxWidth and Stack. All verified: TypeScript clean, ESLint clean, dev server HTTP 200.

---

**Phase 3.4 — Navigation System**
Status: Completed

Built the scroll-aware Navigation system (5 components + 1 hook). Created Navigation types (NavItem, NavigationProps, MobileMenuProps). Created `useScrollDirection` hook with threshold-based direction detection and passive scroll listeners (requestAnimationFrame-level debouncing). Created Navigation component with scroll-aware show/hide behavior (hides on scroll down, reveals on scroll up), responsive layout (mobile hamburger → desktop links), and smooth transitions. Created NavLink component with active route detection via TanStack Router. Created MobileMenu component with slide-in overlay, focus trap, and escape key handling. Fixed TypeScript errors in `use-scroll-direction.ts` (ref access patterns). Fixed `tsconfig.node.json` `allowImportingTsExtensions` error. Fixed ESLint `vite.config.d.ts` parsing issue. All verified: TypeScript clean, ESLint clean, build passes.

---

**Phase 3.5 — Theme Integration**
Status: Completed

Built the global theme infrastructure (16 deliverables). Created ThemeContext with ThemeMode (light/dark/system), ResolvedTheme, and useTheme hook. Enhanced ThemeProvider with prefers-color-scheme detection via matchMedia, localStorage persistence (sovereign-theme-preference), data-theme attribute on html, Tailwind dark class management, and color-scheme CSS property for native form styling. Integrated reduced-motion detection with Redux sync. Created 6 token files mirroring CSS custom properties: color.ts (light + dark mode palette — 3-role system per DESIGN_SYSTEM §3), typography.ts (two families, six-level scale per §4), radius.ts (5-tier scale per §8), shadow.ts (warm-toned elevation per §9, warm glow for dark mode), motion.ts (durations, easings, transform limits per §14), z-index.ts (8-layer stacking system). Created convenience hooks: useResolvedTheme, useIsDark, useThemeMode, useThemeTokens, useReducedMotionTheme. Created useFontLoading hook tracking Cormorant Garamond and DM Sans via Font Loading API. Created theme utilities: getCssVar, setCssVar, getBackgroundColor, getTextColor, getAccentColor, isDarkMode, enableThemeTransition, prefersReducedMotion, getAnimationDuration, validateThemeTokens. Added dark mode CSS variables to tailwind.css with [data-theme="dark"] selector (warm charcoal surface, warm off-white text, brighter gold accent, warm glow shadows per DESIGN_SYSTEM §16 DM1-DM4). Added theme transition CSS (350ms cross-fade per DM7). Added z-index and motion CSS tokens. Updated portal root z-indexes to use CSS variables. All verified: TypeScript clean, ESLint clean, build passes.
